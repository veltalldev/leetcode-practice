### Problem: First Recurring Character

Given a string, return the first recurring character in it, or `null` if there is no recurring character.

#### Example 1:
- **Input**: `"acbbac"`
- **Output**: `"b"`

#### Example 2:
- **Input**: `"abcdef"`
- **Output**: `null`

### Function Signature
```dart
String? firstRecurringCharacter(String input);
